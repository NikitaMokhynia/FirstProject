JavaScript Может изменить HTML-контент

Одним из многих методов HTML-методов JavaScript является getElementById().

В приведенном ниже примере код находит элемент HTML (с id="demo") и изменяет содержимое элемента (innerHTML) на "Hello JavaScript":

Пример

document.getElementById("demo").innerHTML = "Hello JavaScript";

JavaScript принимает как двойные, так и одинарные ковычки:

Пример

document.getElementById('demo').innerHTML = 'Hello JavaScript';
